Privacy Policy

Effective Date: 2026.03.09
Developer: Aaqib Anwar @ Aaqib's DevDesk

This Privacy Policy shortly explains how information is handled within the Insight College Automated School Bell System.

1. Data Collection
The system may store the following information:
User authentication details (via Netlify Identity)
Bell schedules and period timings
Administrative configuration data
No personal student information is collected by this system.

2. Data Storage
System data is securely stored using:
MongoDB Atlas (Cloud Database)
Netlify Services (Hosting and Backend Functions)
These services maintain their own security protocols and compliance policies.

3. Data Usage
Collected data is used solely for:
Managing bell schedules
System functionality
Administrative operations
Data is not sold, shared, or distributed to third parties.

4. Security
The system implements authentication mechanisms to restrict access to authorized users only.

5. Data Retention
Operational data such as schedules may be stored as long as the system remains in use by the institution.

6. Third-Party Services
The system utilizes the following external services:

Netlify
MongoDB Atlas
HiveMQ MQTT Broker

Each service has its own privacy policies and security practices.
